Making the analysis of median home price in Santa Clara county by zip code with median/mean income and median/mean academic performance index of public schools. Data are from the Census Bureau and California department of education websites and saved in facts.json.

facts.py fetches data and save it to facts.json file.
Plot.py creates plots.
App_Front.py starts the application. 
Data Analysis.docx has a brief summary about the findings from the project. 


This application lets the user to choose a city within Santa Clara county then display zip codes within the chosen city. The user can choose 2-5 zip codes and once they selected all zip code, can click compare button to draw the comparison. From the plot chart, the user can see correlations between data.
